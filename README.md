# Food Ordering Website

A full-stack MERN Food Ordering Website with authentication, cart system, order management, and admin dashboard.

## Features

### User Features
- User Signup & Login
- JWT Authentication using Cookies
- Browse Food Items
- Add to Cart
- Place Orders
- View My Orders
- Responsive UI using Tailwind CSS

### Admin Features
- Admin Dashboard
- Add Food Items
- Delete Food Items
- View All Orders
- Update Order Status

---

# Tech Stack

## Frontend
- React.js
- React Router DOM
- Axios
- Tailwind CSS
- Vite

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Cookie Parser
- CORS

## Cloud
- Cloudinary (Image Hosting)
